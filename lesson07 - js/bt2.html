2. Hãy khai báo các biến và cho biết kết quả của các phép toán dưới đây?
Giải thích vì sao có kết quả đó?
name = 'AAA', age = 20, isGood = true
age + 10 = 30
age % 3 = 2
name + ' BBB' = "AAA BBB"
!isGood = false
name == 'aaa' && age >= 20 → false
name != 'aaa' && isGood → true
!(age < 10) && !isGood → false
